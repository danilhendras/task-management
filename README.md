## Description

(Work in progress) A simple task management webapp built using [TypeScript](https://typescriptlang.org) and [Nest.js](https://nestjs.com).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
